# Trello Automation with Cypress and Cucumber

This project showcases how to automate Trello board operations using **Cypress**, **Cucumber**, and **Trello API**. It covers critical functionalities such as creating boards and cards, managing templates, and verifying UI behavior through end-to-end and API testing.

---

## Key Functionalities

1. Create Board       
2. Delete Board-Api
3. Create Card Template  
4. Create Card 
5. Create List-Api 
6. Delete Card  
7. Update Card Template Name 
8. Move Card Template  
9. Hide Card Template
    
---

## Tech Stack

**Cypress** | **Cucumber** | **JavaScript** | **Trello API**  



## How to Run the Project

1. Clone the repo:
```shell
git clone https://github.com/Ansam56/Trello.git
cd Trello
```
2. Install dependencies:
```shell
npm install
```
3. Create a folder named Ccumber inside the e2e directory, and within it, create another folder named Tests.
    
4. Add your Trello APIKey and APIToken
   
5. Run tests:
```shell
npx cypress open
```
